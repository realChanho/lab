# CLAUDE.md

## 이 폴더는 무엇인가

`agentic-engineering`은 Claude Code, Codex 같은 AI 코딩 에이전트로 바이브 코딩을 잘하기 위한 지식을 모으는 노트 폴더다. 실행 코드 프로젝트가 아니라 자료·요약·기준 문서를 정리하는 곳이다.

## 구조

- `karpathy-claude-code/` — 카파시 CLAUDE.md 4원칙. `CLAUDE.md`는 실제 적용용 규칙 파일, `summary.html`은 출처 영상 요약 노트다.
- `project-doc-structure/` — 새 프로젝트 시작 시 잡을 문서 구조 기준. `index.html`에 CLAUDE.md와 스펙·컨벤션 분리 원칙을 정리했다.

## 작성 규칙

- 산출물은 HTML로 만든다. 기존 문서와 같은 다크 테마를 유지한다.
- 외부 자료를 정리할 때는 출처를 문서 안에 명시한다.
- 한국어 문장은 마침표로 끝낸다. 콜론으로 끝내지 않는다.
- 요청받지 않은 문서를 임의로 생성하지 않는다. 무엇을 만들지 먼저 확인한다.
- 주제별로 폴더를 나눈다. 한 폴더에 섞지 않는다.
